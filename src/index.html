Как изменить отступы внутри элемента?

Правильный ответ: padding - внутренний отступ между содержимым и границей

Какой тег используется для создания ненумерованного списка?

Правильный ответ: ul
<ul> создает ненумерованный список

Какие из этих CSS-свойств НЕ влияют на положение элемента в потоке? (выберите все подходящие)

Правильные ответы: visibility: hidden, opacity: 0, color: red

Что выведет этот CSS код?

div { color: red !important; }
div { color: blue; }
Правильный ответ: Текст будет красным
!important имеет высший приоритет

Какой тег сделает текст жирным и семантически важным?

Правильный ответ: <strong> - семантический тег важности

Какой CSS-селектор имеет наивысший приоритет?

Правильный ответ: Инлайн-стиль имеет наивысший приоритет

Какой оператор используется для строгого сравнения в JavaScript?

Правильный ответ: ===

Какие из этих тегов являются семантическими?

Правильные ответы: <header>, <article>, <h1>
